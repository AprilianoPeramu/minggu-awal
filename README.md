<p align="center">
<img src="f.png"/>
</p>

* HTML adalah bahasa markup standar untuk halaman Web atau lebih sederhananya __kerangka__ untuk membuat webstie.Dengan HTML Anda dapat membuat Website Anda sendiri.HTML mudah dipelajari - Anda akan menikmatinya!

## SIMPLE HTML DOCUMENT

```
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>

        <h1>My First Heading</h1>
        <p>My first paragraph.</p>

    </body>
</html>
```
```
| Syntax | Description |
| ----------- | ----------- |
| Head | Kepala |
| Body | Badan |
| Nav  | Navigation |
| h1 | Heading 1|
| p | Paragraf |
```
Elemen HTML didefinisikan oleh tag awal, beberapa konten, dan tag akhir.

+ - [HTML TAGS](https://www.w3schools.com/tags/default.asp)

## HTML Attributes
1. Semua elemen HTML dapat memiliki atribut
2. Atribut memberikan informasi tambahan tentang elemen
3. Atribut selalu ditentukan di tag awal
4. Atribut biasanya datang dalam pasangan nama/nilai seperti: nama="nilai"
+ - [HTML Attributes](https://www.w3schools.com/html/html_attributes.asp)

## HTML Link
Tautan HTML adalah __*hyperlink*__. Anda dapat mengklik tautan dan melompat ke dokumen lain.Saat Anda menggerakkan mouse di atas tautan, panah mouse akan berubah menjadi tangan kecil.

## HMTL Comment
__Comment__ digunakan untuk menjelaskan kode, dan dapat membantu saat Anda mengedit kode sumber di kemudian hari.
```

```

+ __*CSS*__ adalah bahasa yang kita gunakan untuk menata dokumen HTML atau bisa di bilang __badan__ dari kerangka(HTMl) itu.
CSS menjelaskan bagaimana elemen HTML harus ditampilkan.

## SYNTAX

 __CSS__ memiliki struktur seperti gambar di bawah ini.

<img src="z.gif" width="500"/>

+ Pemilih menunjuk ke elemen HTML yang ingin Anda gaya.
Blok deklarasi berisi satu atau lebih deklarasi yang dipisahkan oleh titik koma.
Setiap deklarasi menyertakan nama properti CSS dan nilai, dipisahkan oleh titik dua.
Beberapa deklarasi CSS dipisahkan dengan titik koma, dan blok deklarasi dikelilingi oleh kurung kurawal.
> Untuk refrensi bisa cek melalui link CSS di bawah ini.

- [CSS](https://www.w3schools.com/cssref/default.asp)

__EXAMPLE:__
```
p {
    color: red;
    text-align: center;
}
```
Kamu juga bisa comment di CSS dengan cara
```
/* This is a single-line comment */
p {
  color: red;
}
```

## ALGORITHM & PSUEDOCODE

+  Algoritma adalah deskripsi berupa step-step yang dibutuhkan untuk menyelesaikan suatu masalah.

__EXAMPLE:__
```
Step 1 : Start
Step 2 : Declare Variabels num1, num2 and sum.
Step 3 :Read Values num1 and num2
Step 4 :Add num1 and num2 and assign the result to sum. sum1<--num1 and num2
Step 5 :Display sum
Step 6 :Stop
```
